# 黎潇说明书 — PRD 规划

> **视觉参考**：基于用户提供的个人主页截图（干净现代卡片风）
>
> **整体风格**：Apple 设计感 × 适度科幻点缀
> - **Apple 核心**：大量留白（margin/padding 奢侈）、极简排版、超高字体对比度、Figma 级别的微交互动效
> - **科幻点缀**：极克制，只在关键处点睛（卡片角部、标题装饰、hover 光晕）
> - **配色**：纯白/浅灰背景 + #1d1d1f 深黑文字 + 低饱和彩色标签（Apple 配色哲学）
> - **字体**：主标题超大（96px+）+ 超细字重（200-300）+ 超大 letter-spacing

## 3. 首页模块设计

### 3.1 Hero 关于我

**布局**：左侧头像区域 + 右侧文字区域，水平排列（PC 端）

**左侧 — 头像**：
- 圆形头像框，边框 3px 纯色（如蓝色）
- 暂无照片时显示占位图（文字：「黎潇」）

**右侧 — 个人信息**：
- 大标题：`黎潇`（96px+ 字号，font-weight 200-300，超大 letter-spacing 0.05em，Apple 标志性超细无衬线体风格）
- 副标题：24px，font-weight 400，颜色 #86868b（Apple 官方灰色副标题色）
- 数字标签（三个）：每个数字 48px font-weight 600，说明文字 14px #86868b，底部 1px 细渐变光条

| 标签 | 示例值 | 说明 |
|---|---|---|
| 10+ | 参与项目数 | 待确认 |
| 50+ | 技术栈标签数 | 按实际技能数填 |
| 20+ | 学习周数 | 待定 |

**下方 — 技能标签组**：首屏直接展示全部技能标签，无需滚动即可看到技能全景

### 3.2 技能清单

**设计**：四列卡片布局，每列一个技术域，标签平铺

```
┌──────────────┬────────────────────┬───────────────┬──────────────┐
│  🎨 前端      │  ⚡ 后端          │   🔮 AI 引擎   │  🛡️ 运维     │
│              │                    │               │              │
│  Next.js     │  Python            │  ML·Scikit    │  Docker      │
│  React       │  C / Java          │  PyTorch      │  K8s         │
│  Tailwind    │  FastAPI           │  大模型·DeepSeek│  GitHub Actions│
│              │  REST API          │  ·Qwen·Ollama │  LangSmith   │
│              │  Streaming         │  Agent·LangChain│  腾讯云       │
│              │  MySQL             │  ·Dify·Coze   │              │
│              │                    │  RAG·Faiss    │              │
│              │                    │  ·Milvus·GraphRAG│            │
│              │                    │  数学·数值分析  │              │
│              │                    │  ·数理统计     │              │
│              │                    │  ·矩阵论       │              │
└──────────────┴────────────────────┴───────────────┴──────────────┘
```

**视觉规范**：
- 每列一个卡片（纯白背景，无阴影或极轻微阴影，Apple Card 质感）
- 标题行：emoji + 分类名 + 左侧极细渐变竖线（1px，颜色对应分类）
- 卡片整体：极轻描边（1px #e5e5e7）+ 12px 圆角（Apple 圆角体系）
- 技能标签：圆角胶囊形（pill badge），背景极低饱和度（#f5f5f7 灰蓝/ #fff7f0 橙/ #f5f0ff 紫/ #f0fff5 绿），文字 #1d1d1f
- 悬停效果：标签背景加深 + 轻微上浮 translateY(-2px)，过渡 0.2s ease
- 科幻点缀：卡片右上角有极小装饰性切角（clip-path），不喧宾夺主

### 3.3 项目作品

**设计**：2×2 网格布局（PC 端），每张卡片结构相同

**卡片内容**：
- 项目封面图（16:9 比例，颜色渐变占位图，标注「项目 N」）
- 项目名称（留空，用户后续填写）
- 一句话简介（留空，用户后续填写）
- 技术标签（用户后续填写）

**初始状态**：四个卡片均为「空位」状态，纯白背景 + 中心小字「即将上线 · 项目 N」（16px，颜色 #86868b），四角极轻微圆角（12px），整体 Apple Card 质感

## 4. 技术选型与设计系统

### 4.1 技术栈

| 层级 | 技术选型 |
|---|---|
| **框架** | Next.js 14+（App Router） |
| **样式** | Tailwind CSS |
| **字体** | Inter（最接近 Apple San Francisco 的 Google Fonts 选项，font-weight 按需使用 200/300/400/600） |
| **图标** | Lucide React（线条图标，Apple 风格一致） |
| **部署** | GitHub Actions → 腾讯云 Nginx |
| **域名** | 需确认（待备案） |
| **HTTPS** | Let's Encrypt（腾讯云免费证书） |

### 4.2 设计系统（Apple Design Tokens）

| 类别 | 值 |
|---|---|
| 背景主色 | `#ffffff` |
| 背景次色 | `#f5f5f7` |
| 文字主色 | `#1d1d1f` |
| 文字副色 | `#86868b` |
| 描边色 | `#e5e5e7` |
| 卡片圆角 | `12px` |
| 胶囊圆角 | `9999px` |
| 过渡动画 | `0.2s ease` |
| 最大内容宽度 | `980px`（Apple 官网标准） |
